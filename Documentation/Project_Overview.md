# Project Overview

## Project Name
Customer Retention & Attribution Analysis Using MySQL

## Overview

This project analyzes customer purchasing behavior, customer retention, and marketing attribution using MySQL. The objective is to understand how customers return after their first purchase, identify customer drop-offs, measure repeat purchase behavior, and evaluate the effectiveness of different marketing channels.

The project uses advanced SQL concepts such as Common Table Expressions (CTEs), Window Functions (LEAD, LAG, RANK, DENSE_RANK, ROW_NUMBER), Cohort Analysis, and Attribution Modeling.
